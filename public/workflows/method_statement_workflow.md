# WF-02 — Method Statement Review Workflow (MS)

Trigger: Contractor submits Method Statement
Output: Approved / Revise / Rejected MS

---

## STAGE 1 — COMPLETENESS CHECK

Required sections in MS:
- [ ] Scope of work (clear, matches contract scope)
- [ ] Reference drawings (number, revision, date)
- [ ] Materials list (each with valid MAR reference number)
- [ ] Equipment list (type, capacity, calibration if required)
- [ ] Step-by-step execution sequence
- [ ] Pre-conditions / prerequisite works
- [ ] QA/QC controls (hold points, witness points, checks)
- [ ] Inspection stages cross-referenced to ITP
- [ ] Safety and environmental controls
- [ ] Risk assessment
- [ ] Responsible persons / supervision level

IF any missing → INCOMPLETE (Code I). Return immediately.

---

## STAGE 2 — SPEC COMPLIANCE CHECK
Agent: Spec Compliance Agent

Check method against specification requirements:
1. Execution sequence — does it comply with spec sequence requirements?
2. Environmental conditions (temperature, humidity, curing conditions)
3. Application rates, thicknesses, spacing
4. Equipment requirements (spec may mandate specific equipment)
5. Curing / protection requirements
6. Testing during execution (e.g. compaction testing frequency)
7. Surface preparation requirements before application

---

## STAGE 3 — TECHNICAL REVIEW
Agent: Submittal Review Agent

Critical checks:
- Are all materials listed in MS covered by valid approved MARs?
- Is the execution sequence logical and constructable?
- Are QA controls sufficient to catch deviations during execution?
- Are hold points placed at correct critical stages?
- Is safety adequate for the risk level of the work?
- Does sequence account for weather / environmental constraints?
- Are interfaces with adjacent works addressed?

---

## STAGE 4 — DOCUMENT GENERATION (if gaps found)
Agent: Document Generator

If MS is partially acceptable but missing QA section:
→ Generate QA controls section as recommended addition
→ Generate missing ITP if not yet submitted with MS

---

## STAGE 5 — DECISION

ALL sections present + compliant → CODE A — APPROVED
Minor gaps (QA comments) → CODE B — APPROVED WITH COMMENTS
Missing sections / non-compliant sequence → CODE C — REVISE
Fundamental safety or spec issue → CODE D — REJECTED

---

## STAGE 6 — SITE ACTIVATION CHECK

Before work starts (even with approved MS):
- [ ] MS issued to site supervisor
- [ ] Site team briefed on MS (toolbox talk record)
- [ ] Materials on site and MIR complete
- [ ] Equipment on site and calibrated
- [ ] Pre-conditions verified (prior work approved)
- [ ] ITP distributed and acknowledged

HOLD: Work cannot start without all pre-conditions met.

---

## RESPONSE TIMELINE
Review: 7 days
Resubmission: 5 days
