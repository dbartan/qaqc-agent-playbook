QAQC AI Agent Playbook

Faz bazlı (Earthwork, Shell&Core, Fitout, Snag, T&C, Handover) QAQC yönetimi için Aconex-benzeri agent matrix ve workflow.

Agent Matrix

  • DocuGate: Submission, spec/code/QMP/QCP/ITP uygunluk, revizyon ve comment yönetimi
  • FieldFlow: Pre/During/Post inspection, tolerans, hold/witness point, embedded MEP kontrolü
  • PhaseSync: WBS + real-time phase readiness, "cannot start" blokajları
  • InsightCore: KPI, red flag, RCA, üst yönetim raporu

Phase Akışı

  1. Earthwork
  2. Shell & Core
  3. Fitout
  4. Snag / De-snag
  5. Test & Commissioning
  6. Handover & Closeout

Doküman Kontrol Mantığı

Her faz için Document Gate → Field Gate → Test Gate → Compliance Gate → Reporting Gate sırası uygulanır.
